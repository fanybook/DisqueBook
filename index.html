Disque 使用教程（DisqueBook.com）¶

Disque 是一个内存储存的分布式任务队列实现， 它由 Redis 的作者 Salvatore Sanfilippo (@antirez)开发， 目前正处于预览版（alpha）阶段。

本文档将对 Disque 的安装方法和运行方法进行介绍， 说明各个 Disque 命令的作用， 并给出各个命令的运行示例， 帮助读者更好地理解 Disque 的使用方法。

关于 Disque 项目的更多信息可以参考 Disque 项目的 GitHub 页面： github.com/antirez/disque 。
